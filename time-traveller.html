Time Traveller

2D Time Manipulation Platformer

Overview

Time Traveller is a 2D puzzle-platformer prototype centered around time manipulation mechanics and structured environmental interaction. The project focuses on introducing a unique gameplay mechanic while maintaining clarity and progressive learning.

▶ Watch Gameplay on YouTube

Core Gameplay Loop

Explore → Manipulate Time → Solve Environmental Puzzle → Progress

Each level introduces new constraints or variations of the time mechanic, encouraging experimentation and strategic thinking.

Design Decisions

The time manipulation mechanic was introduced gradually to prevent player confusion. Visual cues and object state feedback were designed to clearly communicate time changes and their impact on the environment. Puzzle difficulty was scaled based on mechanic familiarity rather than complexity alone.

Technical Implementation

• Implemented time-state tracking system • Designed object behavior switching logic • Built conditional triggers for puzzle mechanics • Structured level progression for mechanic reinforcement • Optimized player-state transitions for smooth interaction

Challenges & Solutions

Ensuring clarity during time manipulation was the biggest challenge. Early versions caused confusion due to insufficient visual feedback. This was resolved by refining object state indicators and improving animation transitions.

What I Learned

This project strengthened my understanding of mechanic introduction, player guidance, and puzzle-based progression design. It reinforced the importance of clarity when designing unconventional gameplay systems.
