Board Game

Turn-Based System Design

Overview

Board Game is a structured turn-based physical strategy game focused on rule clarity, balanced player interaction, and decision-driven progression. The project explores system design beyond digital implementation, emphasizing fairness and strategic depth.

Game Visuals

Core Gameplay Loop

Roll → Move → Trigger Event → Resolve Outcome → Switch Turn

The gameplay loop encourages planning, probability assessment, and tactical decision-making within a structured rule framework.

Design Decisions

Rules were intentionally structured to prevent ambiguity and dominant strategies. Player advantages and progression mechanics were balanced through multiple playtesting iterations. The design emphasizes fairness, clarity, and strategic depth.

System Structure

• Defined turn-based state transitions • Designed event-trigger framework • Balanced reward and penalty mechanics • Structured win-condition logic • Iteratively refined rules through testing

Challenges & Solutions

Early versions revealed balance issues where certain strategies dominated gameplay. Adjustments to rule constraints and event probabilities created a more competitive and fair experience.

What I Learned

This project strengthened my understanding of system-based design, balance iteration, and rule clarity in structured gameplay experiences.
