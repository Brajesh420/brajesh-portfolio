Stack The Blocks

2D Physics-Based Mobile Arcade

Overview

Stack The Blocks is a 2D physics-based mobile arcade game focused on precision timing and progressive difficulty scaling. The project explores clean mechanic design, short-session engagement, and consistent physics behavior for mobile platforms.

▶ Watch Gameplay on YouTube

Core Gameplay Loop

Tap → Drop Block → Stack Precisely → Score → Increase Speed → Repeat

The gameplay loop is designed for quick mobile sessions, with difficulty increasing gradually to maintain engagement without overwhelming the player.

Design Decisions

Physics behavior was intentionally tuned to remain predictable rather than random. Difficulty scaling was implemented through movement speed adjustments instead of sudden mechanic changes. The design emphasizes clarity, consistency, and player mastery.

Technical Implementation

• Implemented physics collision tuning for stable stacking • Designed score tracking and progression system • Built difficulty increment logic based on successful stacks • Optimized performance for mobile devices • Structured modular code for mechanic expansion

Challenges & Solutions

Maintaining consistent stacking behavior required fine-tuning physics parameters. Early versions resulted in unpredictable block behavior. Adjusting gravity scaling and collision properties ensured reliable stacking mechanics.

What I Learned

This project improved my understanding of physics tuning, mobile optimization, and short-loop engagement design. It reinforced the importance of predictability in skill-based arcade mechanics.
